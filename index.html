Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is small and lightweight. Inside a host environment (e.g., a web browser), JavaScript can control objects like the Document Object Model (DOM).

What you should already know

This guide assumes you have basic knowledge of the Internet, HTML, and some programming experience.

JavaScript and Java

JavaScript and Java share similarities, but JavaScript has dynamic typing and a prototype-based object model, while Java uses static typing and a class-based object model.

Hello world

To get started with writing JavaScript, write a simple function that outputs a message to the browser:

function greetMe(yourName) { alert("Hello " + yourName); }

Call the function using:

greetMe("World");
Variables

In JavaScript, variables represent values in your program. They must be declared before use and can be assigned values.

Declaring variables

Variables can be declared with `var`, `let`, or `const`:

var x = 42; let y = 13; const PI = 3.14;
Variable scope

Global variables are accessible throughout the document, while local variables are confined to the function or block they are declared in.

Global variables

Global variables are properties of the global object (in a web browser, this is the window object).

Constants

Use `const` to create a constant that cannot be reassigned.

const PI = 3.14;
Data types

JavaScript defines several data types, including `Boolean`, `null`, `undefined`, `Number`, `String`, `Symbol`, and `Object`.

if...else statement

The `if...else` statement is used to execute code based on a condition.

if (condition) { statement_1; } else { statement_2; }
while statement

The `while` loop executes a block of code as long as the specified condition is true.

while (condition) { statement; }
Function declarations

A function declaration defines a reusable block of code.

function greet(name) { return "Hello, " + name; }
